# Drip-lite Feature Scope

> "A simple job & estimate tracker so you don't lose work."

## The Test

**A solo painter can understand it in 30 seconds and use it daily with zero training.**

If it fails that test, it's not Drip-lite.

---

## ‚úÖ VISIBLE (Core Value)

### 1. Board View (THE PRODUCT)
- Fixed stages: New ‚Üí Quoted ‚Üí Scheduled ‚Üí In Progress ‚Üí Done ‚Üí Paid ‚Üí Archive
- Drag cards between columns
- One glance = clarity
- **This is what we sell.**

### 2. Basic Job Card
Only these fields:
- Customer name
- Address  
- Job name
- Status
- Notes
- Scheduled date (simple)

### 3. Estimates (Simplified)
- Square footage input
- Rate (from settings)
- Auto-calculated total
- Accept via public link
- **No line item complexity**

### 4. Copy-to-Text (HUGE WIN)
- "Copy estimate link"
- "Copy estimate message"  
- "Copy reminder message"
- "Copy payment link"
- **This is secretly one of our best features.**

### 5. Invoices (Basic)
- Create invoice from job
- Send payment link
- Mark as paid (manual)
- **Do NOT position as "invoicing software"**

### 6. Settings (Minimal)
- Company name only
- Estimating rate ($/sqft)
- That's it.
